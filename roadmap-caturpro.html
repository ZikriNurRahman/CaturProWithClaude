Apa itu CaturPro?
Konsep aplikasi yang akan kita bangun

CaturPro adalah aplikasi mobile untuk latihan pembukaan catur menggunakan metode Spaced Repetition — sistem yang sama dipakai Anki dan Duolingo. User dilatih mengingat urutan langkah pembukaan catur secara efisien, bukan dengan menghafal paksa tapi dengan pengulangan yang terjadwal secara cerdas.

Referensi utama kita adalah ChessReps.com — tapi kita buat versi Indonesia: Bahasa Indonesia, harga Rupiah, payment lokal (QRIS/GoPay/OVO), dan konten yang relevan untuk komunitas catur Indonesia.

📊
Kenapa Ini Peluang Bagus?
Analisis pasar Indonesia
  • PERCASI punya jutaan anggota aktif di seluruh Indonesia
  • Belum ada aplikasi serupa dalam Bahasa Indonesia
  • Komunitas catur online Indonesia sangat aktif di media sosial
  • Catur masuk kurikulum beberapa sekolah di Indonesia
  • Chess.com dan Lichess tidak punya fitur spaced repetition untuk pembukaan
  • Harga premium bisa disesuaikan daya beli Indonesia (lebih murah dari ChessReps)

Tech Stack

Semua teknologi yang kita pakai dan alasan kenapa dipilih

💡 Semua tech stack ini dipilih karena gratis untuk mulai, dokumentasi bagus, dan cocok untuk solo developer. Mas Zikri sudah familiar dengan React Native & TypeScript jadi kita tidak perlu belajar dari nol.
📱
Frontend — Mobile App
Yang user lihat dan sentuh
React Native + Expo
Framework Utama
Satu kode untuk iOS & Android. Expo mempermudah build dan deploy tanpa perlu Mac untuk Android.
TypeScript
Bahasa Pemrograman
Wajib dari awal. Mencegah bug, autocomplete lebih baik, kode lebih mudah di-maintain.
Expo Router
Navigasi
Navigasi berbasis file seperti Next.js. Lebih simpel dari React Navigation untuk project baru.
NativeWind v4
Styling
Tailwind CSS untuk React Native. Styling cepat tanpa StyleSheet yang verbose.
♟️
Chess Engine & Logic
Otak dari aplikasi catur kita
chess.js
Logic Catur
Library JavaScript paling populer untuk validasi langkah catur, deteksi skak mat, generate FEN, dan semua aturan catur.
react-native-chessboard
Tampilan Papan
Komponen papan catur siap pakai untuk React Native. Mendukung animasi dan gesture drag-and-drop.
Algoritma SM-2
Spaced Repetition
Algoritma yang sama dipakai Anki. Kita implementasi sendiri sebagai pure function — mudah di-test dan tidak butuh library tambahan.
🗄️
Backend & Database
Tempat data disimpan dan diproses
Supabase
Database + Auth + Storage
PostgreSQL gratis sampai 500MB. Sudah include auth, realtime, dan storage. Tidak perlu buat backend sendiri.
expo-secure-store
Local Storage Aman
Simpan session token dengan enkripsi. Lebih aman dari AsyncStorage yang tidak terenkripsi.
Zod
Validasi Data
Validasi data dari Supabase di runtime. Mencegah crash karena data tidak sesuai ekspektasi.

Alur Data Aplikasi
Bagaimana data mengalir dari user ke database
1. User buka aplikasi
Expo Router cek session di expo-secure-store. Kalau ada session valid → langsung ke Home. Kalau tidak ada → ke halaman Login.
2. User pilih kursus pembukaan
openingService.ts fetch data kursus dari Supabase. Data divalidasi dengan Zod schema. Disimpan di Zustand store.
3. User mulai drill
useOpeningDrill hook ambil posisi berikutnya berdasarkan jadwal SM-2. chess.js validasi setiap langkah user.
4. User jawab benar/salah
Algoritma SM-2 hitung interval review berikutnya. Progress disimpan ke Supabase secara async.
5. User akses fitur premium
useSubscription hook cek status premium dari Supabase. Kalau belum premium → tampilkan PaywallModal dengan opsi pembayaran Midtrans.

Struktur Database Supabase
Tabel-tabel yang kita butuhkan
SQL — Tabel Utama
-- Tabel user (extend dari auth.users Supabase)
profiles
├── id          uuid (FK ke auth.users)
├── username    text
├── avatar_url  text
├── is_premium  boolean (default false)
├── premium_until timestamp
└── created_at  timestamp

-- Kursus pembukaan catur
courses
├── id          uuid
├── title       text        -- "Sicilian Defense"
├── title_id    text        -- "Pertahanan Sicilian"
├── description text
├── difficulty  text        -- pemula/menengah/lanjut
├── is_premium  boolean
├── eco_code    text        -- kode ECO catur (B20, dll)
└── thumbnail   text

-- Node/posisi dalam pohon pembukaan
opening_nodes
├── id          uuid
├── course_id   uuid (FK ke courses)
├── parent_id   uuid (nullable, FK ke diri sendiri)
├── fen         text        -- posisi papan
├── san         text        -- langkah (e4, Nf3, dll)
├── explanation text        -- penjelasan Bahasa Indonesia
└── order_index integer

-- Progress user per node (untuk SM-2)
user_progress
├── id           uuid
├── user_id      uuid (FK ke profiles)
├── node_id      uuid (FK ke opening_nodes)
├── repetitions  integer    -- berapa kali sudah diulang
├── ease_factor  float      -- default 2.5
├── interval     integer    -- hari ke review berikutnya
├── next_review  timestamp
└── last_reviewed timestamp

-- Riwayat transaksi
transactions
├── id           uuid
├── user_id      uuid
├── amount       integer    -- dalam Rupiah
├── plan         text       -- bulanan/tahunan
├── status       text       -- pending/success/failed
├── midtrans_id  text
└── created_at   timestamp

Fitur Aplikasi

Detail semua fitur yang akan dibangun — free dan premium

🆓
Fitur Gratis
Cukup menarik untuk bikin user ketagihan
  • 3 Kursus Pembukaan Gratis
    Contoh: e4 dasar, d4 dasar, dan satu pembukaan populer. Cukup untuk user rasakan manfaatnya.
  • Mode Drill Harian
    Maksimal 20 kartu per hari untuk user gratis. Cukup untuk latihan rutin tapi ada batasan yang mendorong upgrade.
  • Progress Tracking Dasar
    Lihat berapa posisi sudah dikuasai, streak harian, dan total waktu latihan.
  • Papan Catur Interaktif
    Papan catur penuh dengan drag-and-drop, highlight langkah legal, dan notasi standar.
  • Penjelasan Bahasa Indonesia
    Setiap langkah ada penjelasan kenapa langkah itu dimainkan — dalam Bahasa Indonesia yang mudah dipahami.
⭐
Fitur Premium
Yang bikin user mau bayar
  • Akses Semua Kursus (20+ pembukaan)
    Sicilian Defense, King's Indian, Queen's Gambit, Ruy Lopez, dan banyak lagi. Semua dengan penjelasan Bahasa Indonesia.
  • Drill Tidak Terbatas
    Tidak ada batasan jumlah kartu per hari. Latihan sepuasnya sesuai jadwal SM-2.
  • Statistik Detail & Grafik
    Grafik progress per minggu, heatmap aktivitas, breakdown per pembukaan, dan prediksi kapan akan menguasai suatu pembukaan.
  • Mode Ujian
    Test diri sendiri tanpa hint. Hasil ujian masuk ke perhitungan SM-2 untuk review yang lebih akurat.
  • Sync Multi-Device
    Progress tersimpan di cloud. Ganti HP tidak kehilangan data latihan.
  • Kursus Baru Setiap Bulan
    Subscriber premium dapat akses kursus baru yang dirilis setiap bulan tanpa biaya tambahan.
🔮
Fitur Masa Depan (v2.0)
Setelah aplikasi sudah punya user dan revenue
🤖 Analisis dengan AI
Upload partai catur kamu, AI kasih rekomendasi pembukaan mana yang perlu dipelajari berdasarkan kelemahan kamu.
👥 Leaderboard
Kompetisi streak harian antar user. Siapa yang paling konsisten latihan minggu ini.
🏆 Tournament Mode
Turnamen mingguan khusus subscriber premium dengan hadiah merchandise atau diskon.
📹 Video Penjelasan
Video singkat Bahasa Indonesia untuk setiap pembukaan. Bisa kolaborasi dengan konten kreator catur Indonesia.
